# Loan_Default_Risk_Analysis

This project analyzes loan data to understand the factors that contribute to loan defaults. It includes data cleaning, exploratory data analysis (EDA), and visualizations to identify patterns and trends in borrower behavior.

## 📌 Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Statistical Summaries
* Data Visualization using Matplotlib and Seaborn
* Loan Default Risk Analysis

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📂 Files

* `Loan_Default_Risk_Analysis.ipynb` – Complete analysis notebook
* `loan_data.csv` – Dataset used for analysis
* `README.md` – Project documentation
